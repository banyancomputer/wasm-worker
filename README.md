# Wasm Hasher 
Simple example of implementing and calling a Web Worker from Wasmable Rust to Browser Javascript

Also implements a lil async Client model for the Web Worker

# Pre Requisites
- yarn
- cargo
- wasm-pack

# Run Wasm Tests

`yarn test`

# Run Web Example

## Install Dependencies

`yarn install`

## Run Dev Server

`yarn dev`

## Build the Web App

`yarn build`
